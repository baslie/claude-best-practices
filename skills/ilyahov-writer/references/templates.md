# Шаблоны (готовые каркасы)

## 12.1. Шаблон короткой новости

```
**Заголовок:**
[Кто] [что сделал] [с чем] [когда / на сколько]

**Лид:**
[Что произошло] [когда/где]. [Кого касается / что меняется]. [Источник].

**Детали:**
- [Факт 1]
- [Факт 2]
- [Срок / условие]

**Контекст:**
[1-2 предложения: что было раньше / почему важно]

**Источник:**
[документ / заявление / ссылка]
```

---

## 12.2. Шаблон новости с объяснением

```
**Заголовок:**
[Кто/что] [действие] [объект] [ключевой параметр]: что изменится

**Лид:**
[Главное изменение + кому важно, 2-3 предложения]

**Что случилось**
[2-3 абзаца с фактами]

**Что изменится**
- [Пункт 1]
- [Пункт 2]
- [Пункт 3]

**Кого касается**
[1 абзац]

**Что это значит**
[Практические последствия для читателя, 1-2 абзаца]

**Что дальше**
[Следующие даты, шаги, ожидаемые решения]
```

---

## 12.3. Шаблон статьи-разбора

```
**Заголовок:**
Что изменится из-за [событие]: сроки, правила, последствия

**Лид:**
[Главное изменение + кому важно]

**Что случилось**
[Факты, хронология]

**Почему это произошло**
[Причины, позиции сторон с источниками]

**Что изменится**
[Конкретные последствия списком]

**Кого касается**
[Группы, на которые влияет]

**Что это значит для читателя**
[Практические действия]

**Что дальше**
[Даты, ожидаемые решения]

**Справка**
[Бэкграунд, если нужен]
```
